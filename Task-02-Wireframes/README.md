#Scenario
An e-commerce brand was losing customers during checkout. The layout was cluttered and users were confused. Task was to redesign the landing page structure.

#What I Did
- Designed low-fidelity wireframes using only black, white, and grey — no colors or images
- Applied Information Architecture principles: Hierarchical, Sequential, and Matrix structures
- Used Z-Pattern layout for the hero section and F-Pattern for product listings
- Placed strategic CTAs: above the fold, mid-page, and end of page
- Added annotations explaining user flow, edge cases, and accessibility notes

#Key Insight
Whitespace is not empty — it is an active design element. 
Reduced cognitive load by limiting product cards per row and making the search bar prominent.
